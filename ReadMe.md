# Webpack Getting Started

## 1. Install Webpack

```bash
npm install webpack webpack-cli --save-dev
```
```bash
yard add webpack webpack-cli --dev
```

## 2. Create a project structure

```
dist/
  index.html
src/
  index.js
webpack.config.js
```

## 3. Create a bundle

```bash
npm install lodash --save
```
```bash
yard add lodash
```
```bash
npx webpack
```
